# WEFL(WE are all inFLuencer) 
> 생각을 기록하는 가치소비 플랫폼

<img src ="https://user-images.githubusercontent.com/22094204/170566261-173039f7-5d82-4c81-9a2a-8248e77328eb.png" width = "300">

위플앱은 생각을 기록하는 가치소비 플랫폼으로
소비를 통해 생각을 키우게 하고,
‘생각 기록하기’ 기능을 제공하여
좋은 생각을 기록하고 공유할 수 있도록 돕습니다
